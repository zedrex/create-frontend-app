## Usage
Run `npx create-frontend-app` and it will create a project folder named `client` with the bootstrap.

## Stack
This bootstrap uses:
- React
- Chakra UI
- React Redux
- Redux Thunk
- React Router
- Custom Components based on Chakra UI
