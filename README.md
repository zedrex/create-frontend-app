## NPM Package
[@zedrex/create-frontend-app](https://www.npmjs.com/package/@zedrex/create-frontend-app)

## Usage
Run `npx @zedrex/create-frontend-app` and it will bootstrap a project folder named `client`.

## Stack
This bootstrap uses:
- React
- Chakra UI
- React Redux
- Redux Thunk
- React Router
